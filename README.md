# THM-Challenges (TryHackMe Writeups)

This repository contains my personal writeups for **TryHackMe** rooms/challenges that I’ve completed so far.  
The goal of these notes is to document my learning, methodology, and commands used during each room.

> Disclaimer: These writeups are for educational purposes only. Please use responsibly and only in legal/authorized environments.

## What You’ll Find in These Notes

Typical content across the writeups includes:

- Enumeration steps (Nmap, directory/file discovery, service checks)
- Exploitation approach (common vectors & reasoning)

- Useful commands, payloads, and tooling references
